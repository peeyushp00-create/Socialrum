# SocialRum

**Content intelligence platform for Indian Instagram & YouTube creators.**

SocialRum helps creators with the hard parts of content: finding what's trending in their niche, analyzing what's working, generating scripts, and turning raw footage into polished, captioned videos.

🔗 **Live:** [socialrum.com](https://socialrum.com)

## Features

- **Trend discovery** — surfaces trending content by niche across Instagram & YouTube
- **Creator analytics** — profile analysis, posting patterns, and competitor tracking
- **AI script generation** — conversational, multi-turn script writing tailored to a creator's voice
- **Video editor** — in-browser captioning, transcription, music sync, and export
- **News aggregation** — multi-source creator-relevant news with one-click script generation

## Tech Stack

**Frontend:** React · TypeScript · Vite (deployed on Vercel)
**Backend:** Node.js · Express (deployed on Render)
**Database:** Supabase · PostgreSQL
**AI & APIs:** Claude · Gemini · Whisper · AssemblyAI · Shotstack · HikerAPI

## Architecture

React SPA (Vercel) → Express API (Render) → Supabase/PostgreSQL, with AI and data services integrated server-side.
